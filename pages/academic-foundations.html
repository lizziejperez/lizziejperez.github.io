Academic Foundations

Montgomery College Coursework Repositories

A curated list of coursework repositories showing foundational CS skills across Java, data structures, and low-level systems programming.

Computer Science I (CMSC203)

  • Repo: MC_CMSC203
  • Language: Java
  • Summary: Coursework repository covering core Java programming fundamentals.
  • Concepts: Classes & Methods, Variables, Strings & Character Processing, Conditionals, Loops, Operators (Logical/Relational), JUnit Testing, Arrays (1D/2D), File I/O, Javadoc

Computer Science II

  • Repo: MC_CMSC204
  • Language: Java
  • Summary: Coursework repository focused on data structures and intermediate Java concepts.
  • Concepts: ArrayList, Exception Handling, Generics (classes/queue/stack), Doubly Linked Lists, Iterators, Comparators, Hashing (tables/codes), Buckets/Chaining, Trees, Graphs

Computer Systems

  • Repo: MC_CMSC216
  • Languages: C, ARM Assembly
  • Summary: Coursework repository covering low-level programming and systems fundamentals.
  • Concepts: Assembly opcodes, Dynamic memory (malloc/free), Data structures in memory, Recursion, C ↔ ARM translation, Processes, Job control, System calls, OS functionality
