Intro to Game and Simulation Development Projects

RPG Project

The assignment was to follow/complete the Creator Kit: RPG tutorial, add a minimum of 5 other houses of various sizes, add another NPC, give the NPC branching dialogue that includes my name and some current event, and create 2 more quests with different inventory items.

2D Platformer Project

The assignment was to follow/complete the Bolt 2D Platformer Tutorial, create a new scene, extend the scene so that it includes at least 10 jumps of varying heights and lengths, and make the new scene load level 2.
