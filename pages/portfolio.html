Cursed Islands (February 2022)

Play Now!

Game Description

This is a 3D platformer game where the player is a treasure hunter on a quest to claim all the gold coins on the Cursed Islands. Find out more details in the GDD and Map document.

Highlights

Create moving paths to other parts of the islands.

Collect apples to increase health.

Avoid ghosts and slime once the curse starts.

Walkthrough

Assignment Requirements

For Building Game Worlds (TECH 290), the assignment was to design and create a 3d platformer that includes: 3rd person player character, "outdoor" environment, moving platforms, collection of items, interactive items, health, timer, hazardous areas, and particles.

For Game Programming (TECH 225), the assignment was to design and create a 3d platformer that includes: a main menu, win-lose condition, a simple Timeline, a health system, visible health, a logical gamespace, and win/lose screens.

Wizard Rescue (March 2022)

Play Now!

Game Description

This is a 3D FPS game where the player is a brave wizard on a quest to save captured wizards being held in a monster infested castle. Find out more details in the GDD and Map document.

Highlights

Use unique collectible wands to cast spells and attack enemies.

Find keys to unlock the castle doors.

Use potions to heal your health and replenish your mana.

Walkthrough

Assignment Requirements

For Building Game Worlds (TECH 290), the assignment was to design and create a 3d fps game that includes: 1rst person player character, NPCs, interior/inside environment, collection of items, interactive items, health, particles, and background audio.

For Game Programming (TECH 225), the assignment was to design and create a 3d fps game that includes: a main menu scene, win-lose condition, ammo inventory, visible health, enemies that chase the player, SFX for item pick up, and a pause menu.
